What it does

A launcher that tells you what's going on

No silent freezes, no guessing whether an update is stuck. Every status LarpStrap shows you is read straight from Roblox, Discord, and your install — live.

Status

Live status, not guesses

Roblox and Discord each get their own status row, so you always know whether you're ready to launch or still waiting on something.

Launcher

Swap your launcher, reversibly

Set LarpStrap as your default Roblox launcher with one click. Change your mind later? Restore the original launcher just as easily.

Updates

Checks for updates on its own

LarpStrap checks for the latest version when it opens and installs updates in the background, so you're never stuck on a stale build.

Discord

Rich Presence, done right

Turn on Discord Rich Presence and friends see what you're playing in real time — no extra setup, toggled from one switch.
